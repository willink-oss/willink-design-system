## Outcome notes (recorded as the rollout lands)

1. **The rollout found a DS bug no in-repo gate could have caught — again.** `@willink-labs/react@1.4.0` shipped its dist without a `'use client'` directive while executing `createContext` at module top level (FormField, [ADR-0015](../adr/0015-formfield-api.md)), crashing `next build` for every RSC consumer. Discovered by the clublink-platform rollout, fixed upstream in **1.4.1** ([#54](https://github.com/willink-oss/willink-design-system/issues/54)): tsup `banner` + treeshake off (rollup 4 silently strips module-level directives) + a permanent dist guard (`packages/react/scripts/check-dist-use-client.mjs`, wired into `pnpm build` so the [ADR-0012](../adr/0012-release-verification-policy.md) Layer 0 CI gate enforces the directive's leading position forever). This is the consumer-pilot pattern paying off exactly as in v1.3 (esperanza/WP pilots → css-tokens DX fixes): real consumers are the only gate that exercises the published artifact in its real runtime.
2. **Both consumers adopted 1.4.1 and deleted their interim shims** — clublink-platform's `ds-client.ts` client-boundary re-export and i-willink.com's seven per-file `'use client'` workarounds are gone; Server Components import `@willink-labs/react` directly again. Consumer-side built CSS was verified unchanged across the patch (byte-identical modulo next/font build hashes).
3. **i-willink.com grew an app-extension foreground scale the DS doesn't ship**: five `fg-*` roles (`--color-fg-strong` / `-emphasis` / `-secondary` / `-subtle` / `-faint`, neutral-mapped with dark flips) covering the text-slate-{300..800} ladder the DS's two-step `fg` / `muted` surface can't express. Flagged as **upstreaming candidates for v1.6** — second consumer demand would confirm.

## Goal

Drive adoption of the v1.2 dark-mode mechanism ([ADR-0013](../adr/0013-dark-mode.md): `data-theme` contract, semantic flip, `prefers-color-scheme` auto mode) across the consumer fleet — deferred out of v1.4 as its own cycle. The DS side is done; this cycle is consumer work plus whatever it shakes loose upstream.
